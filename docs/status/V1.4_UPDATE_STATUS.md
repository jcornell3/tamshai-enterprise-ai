**Testing Philosophy**:
- **TDD (Test-Driven Development)**: RED-GREEN-REFACTOR cycle for service application code

  - GREEN phase: Implement minimum code to make tests pass
  - REFACTOR phase: Improve code quality while keeping tests green
- **90% diff coverage**: Prevents regression while not blocking work on legacy code
- **5-layer security scanning**: CodeQL, npm audit, Gitleaks, tfsec, Trivy
